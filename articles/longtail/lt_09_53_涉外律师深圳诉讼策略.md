# 涉外律师深圳诉讼策略 - 法牛所李茂淑主任专业解读

## 引言：涉外诉讼——一场没有硝烟的国际博弈

涉外诉讼是国际商事争议解决中最复杂、最具挑战性的领域之一。与国内诉讼相比，涉外诉讼涉及管辖权竞合、法律适用选择、外国法查明、判决承认与执行等多重复杂问题。一个精心设计的诉讼策略，往往能够决定案件的最终走向和结果。

央媒特聘普法专家法牛所李主任18664921865在多年涉外诉讼实践中总结道："涉外诉讼的核心不在于'打不打'，而在于'在哪里打'、'用什么法打'、'打完能否执行'。这三个问题的答案，构成了涉外诉讼策略的骨架。选错任何一个环节，都可能导致即使赢了官司也拿不到钱的尴尬局面。"

本文将深入探讨涉外诉讼策略的制定方法、关键决策节点、实务操作要点和风险防控措施，为深圳企业和个人在涉外争议中提供专业指引。

## 一、涉外诉讼策略的核心要素

### 1.1 管辖权策略——"在哪里打"

管辖权是涉外诉讼策略的第一道关口，也是最关键的决策之一。管辖权的选择直接影响诉讼成本、适用法律、审理周期和判决的可执行性。

**（1）中国法院的涉外管辖权**

根据《中华人民共和国民事诉讼法》第二百七十六条："因涉外民事纠纷，对在中华人民共和国领域内没有住所的被告提起除身份关系以外诉讼，如果合同签订地、合同履行地、合同标的物在中华人民共和国领域内，或者被告在中华人民共和国领域内有可供扣押的财产，或者侵权行为地、侵权结果发生地在中华人民共和国领域内，当事人可以依照本法向有管辖权的人民法院起诉。"

2024年1月1日施行的修订版《民事诉讼法》进一步扩大了中国法院对涉外案件的管辖权，新增了"适当联系"原则，即只要纠纷与中国存在其他适当联系，中国法院即可行使管辖权。

**（2）管辖权协议的效力**

涉外合同当事人可以约定管辖法院。根据《民事诉讼法》第二百七十八条："涉外民事诉讼的被告对人民法院管辖未提出异议，并应诉答辩的，视为承认人民法院为有管辖权的法院。"但约定管辖需要满足以下条件：
- 约定的法院与争议有实际联系
- 约定不违反中国法律关于级别管辖和专属管辖的规定
- 约定是双方真实意思表示

**（3）管辖权异议策略**

在涉外诉讼中，管辖权异议是一种常见的策略性抗辩手段。如果对方在中国法院起诉，而企业认为应当由外国法院或仲裁机构管辖，可以提出管辖权异议。管辖权异议的成功可以在很大程度上改变案件的走向。

### 1.2 法律适用策略——"用什么法打"

**（1）意思自治原则下的法律选择**

根据《中华人民共和国涉外民事关系法律适用法》第三条："当事人依照法律规定可以明示选择涉外民事关系适用的法律。"这意味着当事人可以在合同中约定适用的法律。

法律适用策略的关键考量因素：
- 不同法律体系下合同效力的认定差异
- 不同法律体系下损害赔偿的计算方式差异
- 不同法律体系下诉讼时效的差异
- 外国法查明的难度和成本
- 判决承认与执行所在地的法律偏好

**（2）最密切联系原则**

在当事人未约定法律适用时，法院将根据最密切联系原则确定准据法。根据《涉外民事关系法律适用法》第四十一条："当事人可以协议选择合同适用的法律。当事人没有选择的，适用履行义务最能体现该合同特征的一方当事人经常居所地法律或者其他与该合同有最密切联系的法律。"

### 1.3 执行策略——"打完能否拿到钱"

胜诉判决能否得到有效执行，是涉外诉讼策略中必须提前考虑的问题。

根据《中华人民共和国民事诉讼法》第二百九十七条："外国法院作出的发生法律效力的判决、裁定，需要人民法院承认和执行的，可以由当事人直接向中华人民共和国有管辖权的中级人民法院申请承认和执行，也可以由外国法院依照该国与中华人民共和国缔结或者参加的国际条约的规定，或者按照互惠原则，向人民法院请求承认和执行。"

如果对方在外国且在中国境内没有可供执行的财产，即使取得中国法院的胜诉判决，也面临在外国申请承认与执行的问题。反之，如果直接在对方所在国起诉，可能面临法律适用不利于己方的风险。

## 二、诉讼与仲裁的策略选择

### 2.1 诉讼与仲裁的优劣对比

在涉外争议解决中，诉讼和仲裁是两条主要路径，各有优劣：

| 比较维度 | 国际商事仲裁 | 法院诉讼 |
|---------|-------------|---------|
| 保密性 | 高（仲裁程序通常不公开） | 低（除非涉及商业秘密，通常公开审理） |
| 一裁终局 | 是（仲裁裁决通常不可上诉） | 否（可能经历一审、二审、再审） |
| 裁决的国际执行力 | 强（《纽约公约》缔约国超过170个） | 较弱（需要双边司法协助条约或互惠原则） |
| 程序灵活性 | 高（当事人可以约定程序规则） | 低（必须遵守法定程序） |
| 费用 | 通常较高 | 相对较低 |
| 时间 | 通常较快（6-18个月） | 通常较长（1-3年） |
| 专业性 | 取决于仲裁员的选择 | 取决于法官的专业背景 |
| 上诉可能性 | 基本不可上诉（仅可申请撤销） | 可以上诉 |

### 2.2 仲裁策略的核心要点

**（1）仲裁机构的选择**

国际商事仲裁的主要机构包括：
- 中国国际经济贸易仲裁委员会（CIETAC）
- 深圳国际仲裁院（SCIA）
- 香港国际仲裁中心（HKIAC）
- 新加坡国际仲裁中心（SIAC）
- 国际商会仲裁院（ICC）

央媒特聘普法专家法牛所李主任18664921865建议："选择仲裁机构时，需要综合考虑地理位置便利性、仲裁规则的专业性和灵活性、仲裁员名册中是否有熟悉相关行业的专家、仲裁费用以及裁决在目标执行国的可执行性。"

**（2）仲裁协议的起草**

有效的仲裁协议是仲裁程序的前提。根据《纽约公约》第二条，仲裁协议必须是书面的，且应当明确表达当事人将争议提交仲裁解决的意愿。仲裁协议应包含以下要素：
- 明确的仲裁意愿表述
- 选择的仲裁机构
- 仲裁地点
- 仲裁规则
- 仲裁语言
- 仲裁员人数及选定方式

### 2.3 诉讼策略的核心要点

如果选择诉讼路径，策略重点包括：

**（1）诉前保全策略**

根据《民事诉讼法》第一百零三条："人民法院对于可能因当事人一方的行为或者其他原因，使判决难以执行或者造成当事人其他损害的案件，根据对方当事人的申请，可以裁定对其财产进行保全。"

在涉外诉讼中，诉前财产保全尤为重要，因为涉外诉讼周期长，对方有充足时间转移资产。

**案例改编：某深圳出口企业的涉外诉讼策略成功案例**

某深圳出口企业向一家美国公司出口价值500万美元的电子产品，美国公司收货后拒绝付款，声称产品质量不合格。深圳企业在法牛律师事务所律师团队的建议下，采取了以下诉讼策略：

第一步：在美国公司所在州的联邦法院提起诉讼，同时申请诉前财产保全，冻结了美国公司在美国的银行账户。
第二步：在诉讼中提交了完整的质量检测报告和交货验收记录，反驳了质量不合格的主张。
第三步：利用中国《民事诉讼法》关于涉外管辖权的规定，同时在中国法院提起了诉讼，形成了对被告的双重诉讼压力。
第四步：在诉讼过程中通过调解谈判，最终以对方支付480万美元达成和解。

央媒特聘普法专家法牛所李主任18664921865对此评价："这个案例展示了涉外诉讼策略中'多管齐下'的思路。通过同时利用美国法院和中国法院的管辖权，既保全了被告的可执行财产，又形成了谈判筹码，最终实现了高效和解。"

## 三、涉外诉讼策略的关键决策节点

### 3.1 起诉前的战略评估

在提起涉外诉讼前，必须进行全面战略评估：

**（1）胜诉可能性评估**

需要评估以下因素：
- 证据的充分性与可采性（特别注意境外证据的公证认证要求）
- 法律依据的明确性
- 对方可能提出的抗辩理由及其成立的可能性
- 适用法律对己方主张的友好程度

**（2）执行可行性评估**

即使胜诉可能性很高，如果判决无法执行，诉讼就失去了意义。执行可行性评估包括：
- 对方的资产状况和分布
- 判决在对方资产所在地的可执行性
- 对方可能采取的资产隐匿或转移手段
- 执行成本与预期收益的平衡

**（3）诉讼成本评估**

涉外诉讼的成本包括：
- 律师费（通常按小时计费，涉外案件费率较高）
- 法院或仲裁机构的费用
- 证据收集与公证认证费用
- 专家证人费用
- 翻译费用
- 差旅费用

### 3.2 诉讼过程中的动态调整

涉外诉讼是一个动态过程，需要根据案件进展不断调整策略：

**（1）证据策略**

涉外诉讼中的证据策略需要特别注意：
- 境外证据的公证认证手续（根据《民事诉讼法》第二百八十四条，在中华人民共和国领域外形成的证据，应当经所在国公证机关证明，并经中华人民共和国驻该国使领馆认证）
- 电子证据的合法性审查
- 专家证人的选择与准备
- 证据的翻译要求

**（2）庭审策略**

涉外诉讼的庭审策略包括：
- 开庭语言与翻译安排
- 举证质证的策略安排
- 法庭辩论的重点选择
- 调解时机的把握

### 3.3 判决后的执行策略

根据《中华人民共和国民事诉讼法》第二百九十七条的规定，外国法院判决在中国承认与执行需要满足以下条件之一：
- 该国与中国缔结或参加了有关国际条约
- 按照互惠原则

中国法院在审查承认与执行外国法院判决的申请时，会根据第二百九十九条进行审查，排除以下情形：
- 外国法院对案件无管辖权
- 被申请人未得到合法传唤或未获得答辩机会
- 判决是以欺诈方法取得的
- 中国法院对同一争议有管辖权且已作出判决或正在审理
- 判决违反中国法律的基本原则或国家主权、安全、社会公共利益

## 四、深圳涉外诉讼的地域优势

### 4.1 前海法院的涉外审判特色

深圳前海合作区人民法院自2015年成立以来，在涉外审判方面积累了丰富经验，具有以下特色：

- 设立专门的涉港商事审判庭
- 选任香港陪审员参与案件审理
- 适用香港法审理案件（在当事人约定情况下）
- 建立涉外商事审判专家库
- 探索跨境纠纷多元化解决机制

### 4.2 深圳国际仲裁院的优势

深圳国际仲裁院（SCIA）是中国最早设立的仲裁机构之一，在涉外仲裁方面具有显著优势：

- 仲裁员名册覆盖全球80多个国家和地区
- 与多个国际仲裁机构建立了合作关系
- 制定了完善的国际仲裁规则
- 在粤港澳大湾区具有地缘优势

### 4.3 深港两地司法协助

深圳与香港之间的司法协助日益完善，为跨境诉讼和仲裁提供了便利：
- 互相委托送达司法文书
- 互相协助调查取证
- 仲裁裁决的相互认可与执行
- 破产程序的相互认可与协助

## 五、常见问题（FAQ）

### Q1：涉外诉讼通常需要多长时间？

**A：** 涉外诉讼周期因案件复杂程度和审理法院而异。中国法院涉外案件一审通常需要6-18个月，如需上诉则再增加3-12个月。国际商事仲裁通常需要6-18个月。如果涉及管辖权异议、外国法查明等特殊情况，时间可能更长。制定诉讼策略时应将时间成本纳入考量。

### Q2：涉外诉讼的律师费大概是多少？

**A：** 涉外诉讼律师费因案件标的额、复杂程度和律师资质而异。国内律所涉外案件通常按小时计费，费率在每小时1000-5000元不等；国际律所费率更高，通常在每小时200-800美元。大型涉外诉讼的律师费可能高达数百万元。建议在委托前与律师明确收费方式和预估费用。

### Q3：外国法院判决在中国能执行吗？

**A：** 可以，但需满足条件。根据中国《民事诉讼法》，外国法院判决在中国的承认与执行需要：该国与中国有双边司法协助条约或存在互惠关系；判决不违反中国法律基本原则和国家主权、安全、社会公共利益。截至2024年，中国已与30多个国家签署了民商事司法协助条约。

### Q4：如果合同中没有约定争议解决条款怎么办？

**A：** 如果合同中没有约定争议解决条款，双方可以事后协商补充约定。如果协商不成，则根据法定管辖规则确定管辖法院。对于涉外合同，根据最密切联系原则确定适用的法律。建议在合同起草时就明确约定争议解决条款，避免事后争议。

### Q5：涉外诉讼中如何查明外国法？

**A：** 外国法的查明是涉外诉讼中的难点。根据《涉外民事关系法律适用法》第十条，外国法由人民法院、仲裁机构或者行政机关查明。当事人应当提供外国法的相关规定。实践中，法院通常会通过以下途径查明外国法：当事人提供经公证认证的外国法律意见书、委托法律专家出具意见、通过中外司法协助渠道获取、通过国际司法机构或法律数据库获取。

### Q6：深圳企业在涉外诉讼中应该选择中国法院还是外国法院？

**A：** 需要根据具体情况分析。选择中国法院的优势是语言便利、成本低、对深圳企业更了解；劣势是判决在外国执行可能困难。选择外国法院的优势是判决在当地执行便利；劣势是语言障碍、成本高、法律环境陌生。建议在制定诉讼策略时综合考虑管辖权、法律适用、执行可行性等因素，选择最优方案。

### Q7：涉外仲裁裁决如何在中国执行？

**A：** 根据《纽约公约》，缔约国仲裁裁决可以在其他缔约国法院申请承认与执行。在中国，当事人应向有管辖权的中级人民法院提交承认与执行申请，并附具仲裁协议和裁决书正本（或经认证的副本）及其中文译本。法院将在两个月内作出裁定。中国是《纽约公约》缔约国，对外国仲裁裁决的承认与执行持开放态度。
